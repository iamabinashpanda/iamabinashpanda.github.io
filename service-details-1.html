Services

Corporate Services Served

Worker Compensation & General Liability

In my role, I am responsible for maintaining the Policy System by providing data fixes and services tailored to client requirements, ensuring compliance with regular updates and state laws. My tasks include resolving issues in the Policy Administration System (PAS) using Duck Creek Debugging Tools, managing rate changes and manuscript versioning, and implementing data fixes based on client specifications using SQL Server and XQuery. Additionally, I handle requests from database logs that fail at the Duck Creek Server, particularly those not initiated through the Guidewire UI.

Commercial Autos

In this role, I was dedicated to delivering technical production services to an insurance carrier through a broker network. As a developer on the development team, I was responsible for creating Insurance Declaration Forms and addressing data issues, such as fixing premium mismatches in the database. I also participated in daily status calls, providing necessary updates to the business, and handled the design and fixing of forms on the production server.

Dwelling

The project focused on the full development of Dwelling Policies and the creation of automation tools designed to enhance the efficiency of Duck Creek developers at the insurance carrier. In this role, I was involved in developing the Rating Engine for Dwelling Policies 03 and designing and creating various custom automation tools integrated within the Duck Creek System. Additionally, I worked on developing Express 2 & 3 UIs and performed code reviews and fixes using the MS Cop Utility.

Homeowners

The project involved providing services for policies related to HO-2, HO-3, HO-4, and HO-5 lines of business. My responsibilities included creating data and table manuscripts based on documentation provided by architects and senior developers. I developed Express 2 pages for all HO policies and participated in full regression testing. Additionally, I addressed and fixed bugs related to referrals and rules, and created DOC forms by mapping them using the Duck Creek Author.

Security Special For Alarm & Guard And Artisan Contract

The project involves developing a new web application that integrates with the One Shield Policy System. This application allows agents and brokers to input all necessary data on a single page, which is then stored in the database and synchronized with the OS-Policy System. The integration includes rating, forms attachment, and post-bind transactions. In this role, I am responsible for developing XML configuration files to ensure smooth integration between the database and the user interface. I manage tasks assigned through Jira tickets to maintain the system’s functionality and performance. Additionally, I assist senior developers with coding and debugging, validate their work functionally, and provide daily updates during status calls.

Accidental Death & Dismemberment

The project focused on the implementation and development of a new front-end system for Specialty Line insurance policies, involving significant changes to the User Interface. In this role, I implemented Express 3 to create a bilingual (English/Japanese) interface and utilized pre-built manuscripts. I was responsible for customizing the dashboard and developing new quote features for the insurance carrier. Additionally, I addressed UI changes based on client feedback and ensured the User Interface was user-friendly and state-specific.
